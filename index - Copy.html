Online Data Tracking

How is our data being tracked in the web?

  • Browser fingerprinting
    • Our browsers execute a script sent from website, this script sends hashed data back to the web server as our “browser fingerprint” A more detailed explanation of how browser fingerprinting works:
  • This data can be used for targeted ads
  • Session cookies

What are some ways we can prevent or minimize being tracked on the web?

  • Never using the internet
    • This method can be the most effective to completely avoid being tracked, however this might not be the most conventional.
  • Clean browsing history often, including cookies
    • Use privacy browsers in order to keep this persistent, or use incognito mode to prevent the use of browser history and cookies altogether.
  • Use a VPN
    • Use a Virtual Private Network in order to hide your real IP address.
  • Limit the permissions given to web applications.
  • Simply limit the amount of data you give.
